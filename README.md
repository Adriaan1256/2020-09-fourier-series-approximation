# Fourier Series Approximation of Periodic Signals

Course – Practical Assignment  
University of Pretoria  
Completed: September 2020

---

## Project Overview

This project focuses on using Fourier series to approximate different periodic signals. The functions analysed include a sine function, a square wave with a 50% duty cycle, a square wave with a 10% duty cycle, and a sawtooth function.

For each signal, the Fourier series representation was derived and evaluated. The project also determines how many components of the Fourier series are required to produce an accurate approximation of each function. The approximations and signals were plotted using Python.

👉 [View Full Project Report (PDF)](docs/ELI220_Practical_Assignment_1.pdf)

---

## Objectives

- Approximate the function x(t) = sin(2πt) using a Fourier series  
- Approximate a square wave with a 50% duty cycle using a Fourier series  
- Approximate a square wave with a 10% duty cycle using a Fourier series  
- Approximate a sawtooth function x(t) = t using a Fourier series  
- Determine how many Fourier series components are required to accurately represent each function
